Textbook: Computer Networking: A top-down approach (6th Edition), by James F. Kurose and Keith W. Ross, Pearson Addison-Wesley

Course Objectives: The objective of this course is to teach the students why the Internet is designed the way it is and how the Internet operates. The lecture will cover topics include computer network architectures and models, physical media and signaling, data link protocols, medium access control, routing and IP, transport services including TCP/UDP, network applications, local-area and wide-area networks.

Course Topics

The lecture will emphasize on Internet-related protocols. A tentative list of topics are listed as follows.
  1. Introduction
    • Internet and protocols
    • Network architecture and protocols
    • Internet protocol stack, history
    • Internet Edge and Internet Core
    • Introduction to network performance
  2. Application Layer
    • Socket Interface
    • Domain Name Service
    • Email
    • File Transfer Protocol
    • HTTP and the World-Wide Web
    • Peer-to-Peer Applications
  3. Transport Layer
    • Connectionless Transport: UDP
    • Implementing Reliability
    • Sliding Window Protocols
    • Connection-oriented Transport: TCP
    • Congestion Control
  4. Network Layer
    • Virtual Circuits vs. Datagrams
    • Router Operation and Routing Algorithms
    • Internet Protocol (IP)
    • Internet Routing
    • Broadcast and Multicast Routing
  5. Link Layer
    • Error Detection and Correction
    • Overview of M/M/1 Queueing Systems
    • Multi-Access Protocols
    • Ethernet Variants
    • Link-Layer Switching
  6. Wireless Networking
    • 802.11
    • Cellular Internet
    • Mobility Management
